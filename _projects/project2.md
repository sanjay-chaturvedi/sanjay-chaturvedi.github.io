AC current measurement using Arduino Uno requires a current sensor and if the current levels are high then we will require a current transformer to reduce the current values to a safe value. This work aims to measure AC currents up to 30A value and for that we need two things:

 * Current transformer rated 30/5 A
 * Current sensor, ACS712 - 5A
